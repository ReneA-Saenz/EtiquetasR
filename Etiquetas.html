Investigación y Ejemplos de Etiquetas HTML

Etiquetas Básicas

  • <html> ... </html>
    Define el documento HTML y es la raíz de la estructura.
    <html> ... </html>
  • <head> ... </head>
    Contiene metadatos como título, estilos y scripts.
    <head> ... </head>
  • <title> ... </title>
    Define el título que aparece en la barra del navegador.
    <title>Mi Página</title>
  • <body> ... </body>
    Contiene el contenido principal visible de la página web.
    <body> ... </body>
  • <p> ... </p>
    Define un párrafo de texto.
    <p>Este es un párrafo.</p>

    Este es un párrafo.

  • <h1> ... </h6>
    Encabezados jerárquicos, de más a menos importante.
    <h1>Encabezado 1</h1>

    Ejemplo:

    Encabezado 1

    Encabezado 2

    Encabezado 3

  • <br>
    Salto de línea (no tiene cierre).
    Primera línea<br>Segunda línea

    Primera línea
    Segunda línea

  • <a href="URL"> ... </a>
    Crea un enlace a otra página o recurso.
    <a href="https://www.facebook.com/movjuveoficial/?locale=es_LA">Visitar Ejemplo</a>

    Visitar Ejemplo
